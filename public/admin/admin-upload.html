bulk product upload

instructions

please upload a file in JSON or CSV format containing the following fields for each product:

  • product id (optional for new entries)
  • name
  • description
  • category
  • image path (relative path)
  • price

example (CSV format):

product id, name, description, category, image path, price
1, Thalasson 6-Pack, Refreshing hydration, Water, ../assets/images/6-pack.webp, 10.00
2, Thalasson Berry Electrolyte, Berry-flavored hydration, Electrolyte, ../assets/images/berry-electrolyte.webp, 15.00

example (JSON format):

[
    {
        "productId": 1,
        "name": "Thalasson 6-Pack",
        "description": "Refreshing hydration",
        "category": "Water",
        "imagePath": "../assets/images/6-pack.webp",
